Samba服务实现Linux与Windows相互共享资源

发表于2020-02-17|更新于2020-02-17
|阅读量:

#什么是samba?

**samba服务与FTP NFS**服务一样，是一种提供网络文件共享的服务；
**samba服务可以让Windows**机器访问搭建在Linux服务器上的共享文件；
**samba服务支持兼容Windows共享，samba服务搭建在底层是Linux操作系统服务器上，而Windows桌面客户端可以通过samba服务访问Linux**服务器上的共享文件，对共享文件数据来说是安全的！
**samba**服务是基于内核级支持的，是个文件系统；也能把远程samba共享出来的资源挂载到本机；
